# psy120l-final-report
Everyday Moments and the Mind
* This repository contains the published HTML version of my PSY 120L final reproducible report.
* This project examined whether a brief outdoor break improved working memory performance compared to a brief indoor break, and whether this effect differed by perceived stress level or sex.
* The report was created using Quarto and R.
* The published report file is index.html.
* The dataset is not included in this repository
